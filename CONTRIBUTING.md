# Contributing

Thanks for taking time to contribute.

## Workflow

1. Fork and create a feature branch off `main`
2. Run the project test suite locally
3. Ensure brand-lint and CBOM checks pass (CI runs both)
4. Open a pull request describing the change and rationale

## Standards

- Python: ruff + mypy clean, type hints required
- Coverage: 85% minimum where tests apply
- Commit format: `type(scope): description`
- Public artifacts must follow brand rules — see [docs/BRAND-RULES.md](docs/BRAND-RULES.md)

## Reporting issues

Use the issue templates. For security vulnerabilities, see SECURITY.md.
